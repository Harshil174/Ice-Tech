Ice Tech – Animated Hero Landing

Tech Stack: React.js, TailwindCSS, Framer Motion, TypeScript, Vite

Installation

  1. Extract the downloaded ZIP file.
  2. Navigate to the project folder ice-tech.
  3. Install dependencies by running npm install.
  4. Start the development server with npm run dev.

Usage

The project includes an animated landing section featuring video background, motion text, and styled overlays. You can modify content by editing the components in src/components/Home.

File Structure

  • src/components/Home/ – Contains main layout components (HeroText, BoatVideo, MapImages)
  • src/assets/ – Static images and video
  • public/ – Public assets

Customization

Edit text and visuals by modifying the respective JSX/TSX files in src/components/Home/.

Browser Support

  • Firefox (latest)
  • Edge (latest)
  • Safari (latest)
